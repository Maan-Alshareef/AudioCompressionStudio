# Audio Compression Studio

A C# Windows Forms desktop application for audio compression and decompression.

## Features
- Load audio files using Browse or Drag & Drop
- Preview original audio before compression
- Display audio metadata
- Compress audio using:
  - Nonlinear Quantization μ-law
  - Delta Modulation
  - DPCM
- Decompress .cmp files and save recovered audio as WAV
- Show progress bar and performance charts
- Cancel processing while running
- Reset original audio values
- Generate compression report

## Technologies
- C#
- Windows Forms
- NAudio
